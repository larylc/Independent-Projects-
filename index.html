About

For a more detailed summary of the projects please visit my GitHub page.

Project Descriptions

Project 1: Summer Research Project 2018 Investigated how movement affects forked fungus beetle social network position.
Project 2: Exploratory Pokemon Analysis 2020 Explored data visualization using Pokemon gaming data.
Project 3: Features of One Gender Schools (2021) Analyzed the key differences between co-ed colleges and single sex colleges.
Project 4: Grandmaster B-days and B-dates (2021) Created original data set of chess grandmaster birth information using web-scraping tools and Wikipedia pages.
Minor Project 1: Understanding Anime Genre Trends (2021) Examined popularity of anime genres.
Minor Project 2: Do Children Have Bad Taste (2021) Explored how tv show ratings differ across age categories.
Minor Project 3: Finding the Missing Records (2021) Created original data set of chess grandmaster rating histories before the year 2000.
Data Visualization App: Pathway to Prodigy (2021) Used original chess prodigy data set to create app that compares the rating trajectories of chess players over age ranges.
